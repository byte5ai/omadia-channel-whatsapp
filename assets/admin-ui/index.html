WhatsApp

Verbinde diesen Bot als verknüpftes Gerät mit einem WhatsApp-Konto. Die Kopplung läuft genau wie WhatsApp Web — einmal QR scannen, fertig.

Lade…
  1. WhatsApp auf deinem Handy öffnen.
  2. Einstellungen → Verknüpfte Geräte → Gerät verknüpfen.
  3. Diesen QR-Code scannen.

Der Code wird automatisch erneuert, falls er abläuft.
